# 在线教育学习平台 python174

## 项目概述

本项目是一款基于Python和Django的在线教育学习平台。它致力于为学校、教师、学生以及院系提供一个全面、便捷的在线学习环境。

## 技术栈

### 前端技术

- Vue2.0
- Element-ui

### 后端技术

- Python3.7
- Django2.0

## 运行环境

- 开发工具：PyCharm + VSCODE
- 数据库：MySQL5.7（或更高版本）
- 数据库管理工具：Navicat10+
- Python版本：Python3.7
- 前端Nodejs版本：14
- 浏览器：谷歌浏览器

## 功能模块

- 注册登录：完成用户注册和登录功能
- 个人中心：个人信息更新
- 学生管理：对学生信息进行管理
- 教师管理：对教师信息进行管理
- 学校资料管理：查询和删除学校资料
- 试卷管理：查询和删除试卷
- 试题管理：对试题进行管理
- 考试管理：对考试进行管理

## 系统角色

- 管理员：负责学校资料、试卷、试题等的管理
- 教师：可发布和查看教学资料，管理学生信息等
- 学生：在线学习，参与考试等
